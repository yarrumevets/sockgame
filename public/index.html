How to play:

  • Press enter to start the game
  • Use the ↑ & ↓ keys to controll your paddle
  • A second player must join in order for score to be counted
  • Additional players will become spectators

Note: This is a proof-of-concet only intended to demonstrate some of the core concepts of a multiplayer web-based real-time game
